# Pet School 宠物学校管理系统

[![Version](https://img.shields.io/badge/version-v1.1.0-blue)](https://github.com/gugaga-giga/pet-school)

## 项目简介

Pet School 是一个基于 Spring Boot + Vue 3 的宠物学校综合管理系统。

## 技术栈

- 后端：Spring Boot 3.4.1 + MyBatis + MySQL + JWT
- 前端：Vue 3 + Vite + ECharts + Axios

## 功能模块

### v1.1.0 新增
- 宠物系统：完整的宠物档案管理

### 现有功能
- 用户认证、宠物医疗、寄养、课程、优惠券、钱包、证书、直播等

## 快速开始

### 后端启动
```bash
cd pet-school
mvn spring-boot:run
```

### 前端启动
```bash
cd pet-school-ui
npm install
npm run dev
```
